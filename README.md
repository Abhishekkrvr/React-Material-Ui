# React Material UI Practice

This repository contains my practice and learning projects using **React** and **Material UI (MUI)**.
The goal of this project is to understand how different Material UI components work and how to use them in real React applications.

## 🚀 Tech Stack

* React
* Material UI (MUI)
* JavaScript

* And many more...

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Abhishekkrvr/React-Material-Ui.git
```

Go to the project folder:

```
cd React-Material-Ui
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
